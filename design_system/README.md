# Your django-cotton x daisyui design system

This app contains the templates which leverage DaisyUI and django-cotton to bring you an easy-to-use component library which you have full ownership over to extend, modify, or delete pieces of.

## Why a Django App?

The only reason this is a django app is so it can ship by default with a template partial for rendering a demo website (conditional on you configuring the static url to your assets).

## Usage

The components are all under `cotton/ds` so that you use them like `<c-ds.button>...</c-ds.button>`.
